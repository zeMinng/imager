# 🎨 imager | 纯前端图像处理工具箱

**imager** 是一款追求极致体验的高性能纯前端图像处理工具。它打破了传统工具对后端的依赖，让“图片不出本地”成为现实。

## 🌟 核心能力

- **🧩 自由拼接**：支持横向、纵向及自定义网格排版，间距、圆角随心调节。
- **🔄 万能转换**：在 JPG、PNG、WebP 甚至 GIF 之间自由穿梭，秒级完成。
- **🗜️ 智能压缩**：基于高性能算法，在保持视觉无损的同时，大幅压低文件体积。
- **✨ 实时预览**：所见即所得，调整参数时实时渲染处理效果。

## 🚀 为什么选择 imager？

| 维度         | 传统在线工具                   | imager (本项目)                        |
| ------------ | ------------------------------ | -------------------------------------- |
| **安全性**   | 图片需上传至服务器，有隐私风险 | **100% 本地处理**，数据不出浏览器      |
| **速度**     | 受限于网速和服务器带宽         | **极速响应**，处理速度仅取决于本地算力 |
| **费用**     | 常见会员制、次数限制或水印     | **永久免费**，无任何套路               |
| **离线使用** | 必须联网                       | 支持 PWA，**断网也能用**               |

## 🛠️ 技术实现

本项目采用前沿的前端技术栈，确保处理性能：

- **计算核心**：利用 Canvas API 与 OffscreenCanvas 实现硬件加速渲染。
- **多线程**：引入 Web Workers 处理复杂压缩算法，避免界面卡顿。
- **格式支持**：集成多种轻量化编解码库。

## 🏃 快速上手

### 1. 环境准备

确保你本地安装了 Node.js 环境。

### 2. 启动项目

```bash
# 克隆仓库
git clone https://github.com/zeMinng/imager.git

# 安装依赖
npm install

# 开启开发服务器
npm run dev

```
